🔌 API ENDPOINTS (20+)
════════════════════════════════════════════════════════════════════════════

Generation (4):
  POST   /api/generate/start              Start infinite generation
  POST   /api/generate/stop               Stop generation
  GET    /api/generate/status             Generation status

Drills (3):
  POST   /api/drill/create                Create deterministic drill

  POST   /api/analyze/image               Analyze one image
  POST   /api/analyze/batch               Analyze many images

  GET    /api/analysis/{image_id}/compare Compare analyses

History (3):

Models (4):

🔧 TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════

Issue: Generation not working
Solution: Check GEMINI_API_KEY environment variable
  echo $GEMINI_API_KEY

Issue: Port 5000 in use
Solution: Kill the process
  lsof -ti:5000 | xargs kill -9

Issue: Database errors
Solution: Delete and recreate
  rm -rf data/red_marker.db
  python run.py

Issue: Vision model missing
Solution: Install the package
  pip install torch torchvision

More help: See QUICK_REFERENCE.md or README_PRO.md
